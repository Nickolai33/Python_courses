
1. Словари – это встроенный тип данных, который является 
ассоциативным массивом или хешем и базируется на отображении пар типа (ключ:значение).

2. Во-первых, с помощью литерала:
   d = {}
   d = {"asd":1,}
   с помощью функции dict:
   d = dict(a='b', v='d')
   d = dict([(1,1),(2,4)])
   В-третьих, с помощью метода fromkeys:
   d = dict.fromkeys(['a', 'b'], 100)
   В-четвертых, с помощью генераторов словарей:
   d = {a: a ** 2 for a in range(7)}

3. В словаре, если известен ключ, то доступ к значению по эту ключу можно получить
 с помощью операции []. В этот же способ можно изменить значение, если известен ключ,
 который соответствует этому значению.

4. dict.clear() - очищает словарь.
    dict.copy() - возвращает копию словаря.
    dict.items() - возвращает пары (ключ, значение).
    dict.keys() - возвращает ключи в словаре.
    dict.pop(key[, default]) - удаляет ключ и возвращает значение. Если ключа нет, возвращает default (по умолчанию бросает исключение).
    dict.values() - возвращает значения в словаре.
    стандартный метод len(dict) – определяет количество элементов в списке.

5.Операция del предназначена для удаления элемента из словаря на основе заданного ключа key.
 Общая форма использования операции, следующая:
 del D[key]

6. Чтобы определить, существует ли заданный ключ в словаре, в Python используется операция in.
   Общая форма использования операции in следующая:

   f_is = key in D

7.Функция zip() позволяет создать словарь путем объединения списков ключей и значений.

8. Как известно, ключи в словаре сохраняются в произвольном порядке. 
Если возникает необходимость отсортировать словарь по ключам, то для этого можно использовать метод sort(),
 который используется для списков. Для этого предварительно нужно конвертировать представление ключей в список.
